
# Instructions

Compile "a.cpp" first

Then run the program "a"

# Windows

`
g++ a.cpp -o a

.\a
`

You can use [CSVFileView](https://www.nirsoft.net/utils/csv_file_view.html) for viewing the .csv files.

It is best to set the "Options > Auto Refresh Mode" to "Full".

# Linux

`
g++ a.cpp -o a

./a
`
